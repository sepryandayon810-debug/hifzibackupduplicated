Permission & Menu — Owner

Belum Disimpan

Centang fitur/menu yang boleh diakses oleh role ini. Perubahan akan berlaku saat user login ulang atau refresh halaman.

Cara Kerja

  • Developer mengatur template permission per role di sini.
  • Data tersimpan di settings/developer/roleAccess/{role}.
  • Saat user login, sistem membaca config ini sebagai master permission.
  • Individual permission di users/{uid}/permissions bisa override master (kalau explicit true/false).
  • Kosongkan individual permission agar user mengikuti master role config.
